## How to use

```bash
# Node version (using nodenv)
$ node -v
v16.13.0

# Install dependencies
$ yarn install --immutable

# Update GraphQL schema (settings in graphql.config.js)
$ yarn schema:update

# Update npm packages
$ yarn package:update

# Run dev server (localhost:4020)
$ yarn dev
```

### Dependencies

```bash
# Global npm packages
$ npm i -g npm yarn
# for relay watch
$ brew install watchman
```
